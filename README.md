# SuperLista

SuperLista é uma aplicação leve e responsiva para organizar suas compras, controlar gastos mensais e reaproveitar itens do histórico.

## Recursos

- Lista de compras com itens pendentes
- Histórico mensal de gastos
- Reutilização rápida de itens já comprados
- Layout responsivo para celular, tablet e desktop
- Armazenamento local no navegador

## Como usar

1. Abra `index.html`
2. Clique em **Abrir Lista de Compras**
3. Adicione o item, quantidade e preço
4. Use o histórico para revisitar compras anteriores

## Estrutura do projeto

- `index.html` — página inicial com acesso ao aplicativo
- `mode2.html` — lista de compras profissional com histórico
- `css/style.css` — estilos modernos e responsivos
- `js/mode2.js` — lógica da lista, histórico e armazenamento local
